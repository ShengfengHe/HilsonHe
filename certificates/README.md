# Certificates

Put Hilson's certificates in this folder as PDF, PNG, or JPG files.

Recommended file names:

- `math-competition-2025.pdf`
- `coding-certificate-2026.pdf`
- `swimming-award-2024.jpg`

After uploading to GitHub, each certificate can be linked from the CV like this:

```markdown
[Math Competition Certificate](certificates/math-competition-2025.pdf)
```

If you need a full public GitHub link, use this pattern:

```text
https://github.com/YOUR-USERNAME/YOUR-REPOSITORY/blob/main/certificates/FILE-NAME.pdf
```
